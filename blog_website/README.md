# Blog Website - Client Project

## Finn Saunders

### Overview:

Custom WordPress theme built from scratch for a client's literary blog. Designed with an editorial minimalist aesthetic and built for a non-technical user to self-publish content through the WordPress admin panel.

### Tech Stack:

- WordPress (custom theme)
- PHP
- Vanilla CSS (with CSS custom properties)
- Vanilla JavaScript
- Docker (local development and deployment)
- MySQL 8.0
- Contact Form 7

### Key Features:

1. Hand-coded WordPress theme with no page builders or pre-built templates
2. WCAG accessibility (skip-to-content link, ARIA attributes, keyboard navigation, focus-visible states, Escape key handling)
3. CSS design system using custom properties for colors, spacing, typography scale, and content width
4. Mobile-first responsive design with animated hamburger menu (3-line to X transform)
5. Editorial typography with 700px content width for comfortable reading
6. Blog index with date, title, excerpt, and pagination
7. Single post template with full content support
8. Docker Compose setup for reproducible development and deployment
